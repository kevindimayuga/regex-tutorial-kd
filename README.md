## Description

The link will lead to a GitHub Gist that will contain a tutorial on the Match an Email RegEx, it's definition and how it used in code.

```
/^([a-z0-9_\.-]+)@([\da-z\.-]+)\.([a-z\.]{2,6})$/
```
